# Store-Sales-performance-Analytics-Excel-Dashboard
---
📊 Project Overview
- This project has a dynamic Dashboard that show the Sales distribution acorss Customers  and ther segmentation , Region , Store_Location , Product and their Category in order to get the full overview From where and how much Sales they are Responsible for 
***

🎯 Objectives
---
- Analysing Sales Monthly to check the any Seasonal Pattern Across Each year via Filter
- Analyzing sales Regions wise Category performance to see which  Category  is performing Across Each Region
- Analyzing Sales Across Category wise  to get the wide View of Categories  and Gender as a Legend
- Analyzing Sales Across AGe group wise and Gender as a Legend to know Which Age_Group is purchasing more and which Gender
- Analyzing  Sales vs Profit Distibution Across the products to get wide of high performing products Vs medium vs Low peforming Products
- Analyzing Each Product's Sales Performance
- Analyzing Sales Across Store Locations to which Store is performing best out of all

***

📈 Dashboard Features
- Analyzed monthly Sales with  line chart
- Analyzed Regions wise Category performance  using heat map Visual
- Analyzed Sales Across Category using Bar chart
- Analyzed Sales Across AGe group wise and Gender as a Legend with Stacked Column charts
- Analyzed Sales vs Profit Distibution Across the products with scatter plot
- Analyzed Each Product's Sales Performance using Bar chart
- Analyzed Sales Across Store Locations with Bar chart

***

🛠️ Tools & Technologies
- Microsoft Excel for the Dashboard
- Used Power Query for Data Cleaning and Tranformation  for Analysis
- used pivot charts and Advance Formulas for Analysis
- used little bit Macros for the Refresh Creation button
